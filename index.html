Reset Default Browser CSS (Visual)

Note:
  • Browser Default CSS should be reset programmatically so no unexpected behavior would occur while viewing on different browser.
  • Basic main attributes like Margin, Padding, box-sizing and font-family should be reset i.e. to assign default value to this attribute.
  • Every element has Browser Default CSS if we want to just reset all of them we can define default style for every element. It will be tedious to do so we can just import an Reset Element Stylesheet from CDN. Although it's not necessary.

Full height of html and body element (Visual)

  • By default HTML & Body element tag does not have full window height. If content of HTML is small than there will be problem for setting footer to bottom of the page.
  • Although if content of HTML is big than windows size then footer will be placed at bottom so this attribute is missed by most of the developers.
